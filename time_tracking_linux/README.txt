Пишем такой код в файл с расширением .sh :
#!/bin/bash
start=$(ps -o lstart 1 | tail -n 1);
echo $start >> starts.txt;
end=$(last -x | grep shutdown | head -1);
echo $end >> ends.txt;


#Если нужно в определенный каталог сохранять:
#echo $d >> /media/admin1/hom1/Coding/Ubuntu/starts.txt;

Важное! Если файл лога в той же директории, то слеш не ставим: echo $end >> ends.txt
Правой кнопкой кликаем на исполняющем файле, заходим в права и ставим галочку: "Разрешить выполнения файла как программы". Или пишем chmod a+x ./название_файла
Потом кидаем исполняющий файл например в Домашнюю папку или куду-то типа диска C:
Потом заходим в Убунте в автоматическую загрузку. Нажимаем добавить, вносим любое имя и описание, потом обзор и ищем исполняющий файл.

Файлы запускаются с помощью автозагрузки: 
start_cli.sh - запускает терминал
start_log.sh - записывает в файл starts_log.txt (общий файл) время старта загрузки компа
             - записывает в файл ends_log.txt (общий файл) время выключения компа 
             - записывает в файл ends_De.txt время выключения компа пользователя (на самом деле ends_log.txt равен ends_De.txt)

Файлы запускаются демоном (crontab):
start_log_wifi.sh - записывает в файл starts_wifi_De.txt старт появления мак адреса гаджета в сети
                  - записывает в файл ends_wifi_De.txt конец появления мак адреса гаджета в сети
#Для запуска нужно зайти в директорию /etc и запустить crontab, прописсав следующее:
* * * * * root /home/admin2/time_tracking/start_log_wifi.sh

Файлы запускаются с помощью пользователя из терминала (./loginDe.sh):
loginDe.sh - записывает старты запуска компьютера конкретного пользователя в файл starts_De.txt
           - записывает старты (время) ввода имени конкретного пользователя в файл starts_input_De.txt
end_input_De.sh - записывает окончание работы компьютера конкретного пользователя в файл ends_input_De.txt
