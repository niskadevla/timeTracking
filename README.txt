1. Открываем директорию "time_tracking_JS"
2. Нажимаем правой кнопкой мыши на пустом месте и выбираем пункт: "открыть в терминале"
3. Пишем http-server(npm install --global http-server) и жмем enter (при условии, что установлено node(sudo apt install nodejs) и npm (sudo apt install npm))
4. Копируем верхний адрес: http://127.0.0.1:8081
5. Вставляем в строку ввода в браузере и жмем enter
6. Потом кликаем файл с расширением *.html
